_____TAMAGOTCHI_____
_______PIPOU________

par Antoine Aribard, Louis Cart-grandjean, Titouan Quintin, Maxence Gautier et Simon Chevé


Prérequis :
	- Java installé sur votre ordinateur


Comment démarrer le jeu :
	- Extraire le dossier Tamagotchi de l'archive .rar
	- Lancer Tamagotchi.jar


Comment commencer une partie :
	- Lancer le jeu
	- Cliquer sur Nouvelle partie
	- Choisir un nom de personnage entre 1 et 20 caractères
	- Choisir le type du personnage
	- Cliquer sur Jouer


Comment charger une partie :
	- Lancer le jeu
	- Cliquer sur le bouton correspondant à la sauvegarde que vous souhaitez lancer
	- Si vous avez plus de 4 sauvegarde, utiliser les boutons sur les côtés pour les parcourir
	- Si aucun bouton de partie à charger ne s'affiche, soit vous n'avez aucune partie sauvegardée, soit le fichier est corrompu


Comment jouer :
	- Votre personnage a des besoins, il a besoin de vous pour les combler
	- Cliquer sur les flèches sur le plan en bas a droite pour vous déplacer(vous pouvez aller seulement dans les pièces adjacentes à la votre)
	- En fonction de la pièce où vous êtes, vous avez accès à différentes actions possibles
	- Les actions(Manger, dormir, etc) font augmenter la valeur dans la barre correspondante(attention, certaines peuvent en faire descendre d'autres légèrement)
	- Si plusieurs caractéristiques ont une valeur élevée, cela fait augmenter la barre de vie
	- Attention, à l'inverse, des valeurs trop basses font baisser la barre de vie, pouvant entraîner la mort du personnage
	- Si la mort de votre personnage fini par arriver, il n'y a pas de retour en arrière possible, la sauvegarde est effacée automatiquement
	- Pour sauvegarder, il suffit d'appuyer sur le bouton correspondant
	- Lorsque vous quittez le jeu, la partie se sauvegarde automatiquement


Nous comptons sur vous pour bien prendre soin de vos petits personnages
